BEV Space Integration（BEV 空间融合模块）¶

BEV Space Integration 模块的作用是：
将来自图像侧与点云侧的 BEV 特征进行统一空间表示与融合，
并通过 BEV 编码器提取高层语义特征，为最终任务检测头提供输入。

本模块包含四类核心功能：

  1. 图像 BEV 特征预处理
    接收 Camera Side 的 BEV 特征并进行维度变换或对齐。

  2. 点云 BEV 特征预处理
    接收 Lidar Side 的 BEV 特征并进行通道、尺度对齐。

  3. 多模态 BEV 融合算子
    将图像 BEV 与点云 BEV 在同一空间中融合（如 concat / add / weighted fusion）。

  4. BEV 编码器（BEV Encoder）
    基于 2D CNN 提取最终高层 BEV 语义特征。

'bev_concat_fusion'¶

将 Camera BEV 与 LiDAR BEV 进行通道维拼接融合。

bev_concat_fusion(bev_cam, bev_lidar, dim=1) ¶

将相机分支与 LiDAR 分支在 BEV 平面上的特征图在通道维度上拼接， 得到融合后的 BEV 特征图，形状为 (B, C_cam + C_lidar, H, W)。

Parameters:
  • bev_cam (Tensor) –

    (B, C_cam, H, W) 相机分支的 BEV 特征图： - B: batch size - C_cam: 相机 BEV 特征通道数 - H, W: BEV 网格的空间尺寸。

  • bev_lidar (Tensor) –

    (B, C_lidar, H, W) LiDAR 分支的 BEV 特征图： - C_lidar: LiDAR BEV 特征通道数 - H, W: 需与 bev_cam 在空间维度上对齐。

  • dim (int, default: 1 ) –

    int 拼接维度，默认为 1（通道维度），通常不需要修改。
Returns:
  • bev_fused( Tensor ) –

    (B, C_cam + C_lidar, H, W) 在通道维度上拼接后的融合 BEV 特征图，可作为后续 BEV 融合编码算子（如 ConvFuser）的输入。

@torch.library.custom_op(
    "bevfusion_ops::bev_concat_fusion",
    mutates_args=[]
)
def bev_concat_fusion(
    bev_cam: torch.Tensor,
    bev_lidar: torch.Tensor,
    dim: int = 1,
) -> torch.Tensor:
    """
    将相机分支与 LiDAR 分支在 BEV 平面上的特征图在通道维度上拼接，
    得到融合后的 BEV 特征图，形状为 (B, C_cam + C_lidar, H, W)。

    Args:
        bev_cam: (B, C_cam, H, W)
            相机分支的 BEV 特征图：
              - B: batch size
              - C_cam: 相机 BEV 特征通道数
              - H, W: BEV 网格的空间尺寸。
        bev_lidar: (B, C_lidar, H, W)
            LiDAR 分支的 BEV 特征图：
              - C_lidar: LiDAR BEV 特征通道数
              - H, W: 需与 bev_cam 在空间维度上对齐。
        dim: int
            拼接维度，默认为 1（通道维度），通常不需要修改。

    Returns:
        bev_fused: (B, C_cam + C_lidar, H, W)
            在通道维度上拼接后的融合 BEV 特征图，可作为后续
            BEV 融合编码算子（如 ConvFuser）的输入。
    """
    assert bev_cam.dim() == 4, "bev_cam 必须是 (B, C, H, W) 张量"         # 检查相机 BEV 形状
    assert bev_lidar.dim() == 4, "bev_lidar 必须是 (B, C, H, W) 张量"     # 检查 LiDAR BEV 形状
    assert bev_cam.shape[0] == bev_lidar.shape[0], "batch 维度不匹配"      # 检查 batch 维
    assert bev_cam.shape[2:] == bev_lidar.shape[2:], "空间尺寸 (H,W) 不匹配"  # 检查空间尺寸

    bev_fused = torch.cat([bev_cam, bev_lidar], dim=dim)                  # 在通道维度拼接特征

    return bev_fused                                                      # 返回融合 BEV 特征

'BEVFusionEncoder'¶

2D BEV 编码器，对融合后的 BEV 特征进行卷积处理。

BEVFusionEncoder ¶

Bases: Module

BEVFusionEncoder 模块（BEV 融合编码算子 / ConvFuser）。

该模块接收在 BEV 平面上拼接后的多模态特征（Camera BEV + LiDAR BEV）， 通过若干层 2D 卷积对通道维与空间维进行联合建模，输出最终的 Fused BEV Features，作为后续检测、分割等任务头的统一输入。

结构说明（按顺序）¶
  1. Conv2d (C_in → C_mid, 3×3) + BN + ReLU
  2. 融合不同模态通道信息
  3. Conv2d (C_mid → C_out, 3×3) + BN + ReLU
  4. 进一步提取 BEV 空间结构特征
  5. （可选）残差连接增强稳定性
输入¶

Args: x: (B, C_in, H, W) 拼接后的 BEV 特征图： - B: batch size - C_in: Camera BEV 通道数 + LiDAR BEV 通道数 - H, W: BEV 网格空间尺寸

输出¶

Returns: fused_bev: (B, C_out, H, W) 融合编码后的最终 BEV 特征图。

模块用途¶
  • 作为 Camera BEV 与 LiDAR BEV 之后的最终融合编码模块
  • 输出的 Fused BEV Features 直接用于下游 3D 检测 / 地图分割等任务

class BEVFusionEncoder(nn.Module):
    """
    BEVFusionEncoder 模块（BEV 融合编码算子 / ConvFuser）。

    该模块接收在 BEV 平面上拼接后的多模态特征（Camera BEV + LiDAR BEV），
    通过若干层 2D 卷积对通道维与空间维进行联合建模，输出最终的
    Fused BEV Features，作为后续检测、分割等任务头的统一输入。

    ---
    ### 结构说明（按顺序）
    1. **Conv2d (C_in → C_mid, 3×3) + BN + ReLU**
       - 融合不同模态通道信息
    2. **Conv2d (C_mid → C_out, 3×3) + BN + ReLU**
       - 进一步提取 BEV 空间结构特征
    3. （可选）残差连接增强稳定性

    ---
    ### 输入
    Args:
        x: (B, C_in, H, W)
            拼接后的 BEV 特征图：
              - B: batch size
              - C_in: Camera BEV 通道数 + LiDAR BEV 通道数
              - H, W: BEV 网格空间尺寸

    ---
    ### 输出
    Returns:
        fused_bev: (B, C_out, H, W)
            融合编码后的最终 BEV 特征图。

    ---
    ### 模块用途
    - 作为 Camera BEV 与 LiDAR BEV 之后的最终融合编码模块
    - 输出的 Fused BEV Features 直接用于下游 3D 检测 / 地图分割等任务
    """

    def __init__(
        self,
        in_channels: int,
        mid_channels: int,
        out_channels: int,
        use_residual: bool = False,
    ):
        """
        Args:
            in_channels (int):
                输入 BEV 特征通道数 C_in。              # 输入通道
            mid_channels (int):
                中间层通道数 C_mid。                   # 中间通道
            out_channels (int):
                输出 BEV 特征通道数 C_out。            # 输出通道
            use_residual (bool):
                是否使用残差连接。                     # 是否启用残差
        """
        super().__init__()  # 调用父类构造

        self.use_residual = use_residual                                 # 记录是否使用残差

        self.conv1 = nn.Conv2d(in_channels, mid_channels, kernel_size=3, padding=1, bias=False)  # 第一层 3×3 Conv
        self.bn1 = nn.BatchNorm2d(mid_channels)                          # BN
        self.conv2 = nn.Conv2d(mid_channels, out_channels, kernel_size=3, padding=1, bias=False) # 第二层 3×3 Conv
        self.bn2 = nn.BatchNorm2d(out_channels)                          # BN

        if use_residual and in_channels != out_channels:                 # 若通道不一致且启用残差
            self.res_conv = nn.Conv2d(in_channels, out_channels, kernel_size=1, bias=False)     # 通道对齐卷积
        else:
            self.res_conv = None                                         # 不使用残差对齐

    def forward(self, x: torch.Tensor) -> torch.Tensor:
        """
        Args:
            x (torch.Tensor):
                输入 BEV 特征图，形状 (B, C_in, H, W)。   # 输入特征

        Returns:
            torch.Tensor:
                融合编码后的 BEV 特征图 (B, C_out, H, W)。 # 输出特征
        """
        assert x.dim() == 4, "x 必须是 (B, C, H, W) 张量"    # 基本形状检查

        identity = x                                       # 保存残差分支

        x = self.conv1(x)                                  # 第一层卷积
        x = self.bn1(x)                                    # BN
        x = F.relu(x, inplace=True)                        # ReLU

        x = self.conv2(x)                                  # 第二层卷积
        x = self.bn2(x)                                    # BN

        if self.use_residual:                              # 若启用残差
            if self.res_conv is not None:                  # 若需通道对齐
                identity = self.res_conv(identity)        # 残差通道对齐
            x = x + identity                               # 残差相加

        x = F.relu(x, inplace=True)                        # 最后一次激活

        return x                                           # 返回 Fused BEV Features
