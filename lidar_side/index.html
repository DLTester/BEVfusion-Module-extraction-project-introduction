Lidar Side（点云侧模块）¶

Lidar Side 模块的作用是：
将原始 LiDAR 点云进行体素化、特征编码，并通过网络提取 3D 语义表示；
随后将 3D 特征压缩至 BEV 空间，为后续 BEV 融合提供点云 BEV 表达。

本模块包含四类核心功能：

  1. 点云体素化（Voxelization）
    将稀疏点云转换为规则体素结构，便于后续网络处理。

  2. 体素特征编码（Voxel Feature Encoding）
    对每个体素内部的原始点特征进行编码，得到固定维度表示。

  3. 稀疏卷积 3D Backbone（Sparse 3D CNN）
    使用稀疏卷积提取 3D 语义特征。

  4. 点云 BEV 压缩算子
    将 3D 特征沿 Z 维压缩成 BEV 特征平面。

'LidarVoxelization'¶

将原始点云转换为稀疏体素结构。

LidarVoxelization ¶

Bases: Module

LidarVoxelization 模块（点云体素化算子）。

该模块将连续的 3D 点云坐标按照给定的 voxel_size 与 point_cloud_range 量化到规则 3D 体素网格中，并输出体素中的点集合、体素坐标以及每个体素的点数。 本实现为简化版 CPU 体素化逻辑，便于阅读与算子级重构，而非高性能实现。

结构说明（按顺序）¶
  1. 根据 point_cloud_range 与 voxel_size 计算 3D 网格大小 (nx, ny, nz)
  2. 遍历 batch 中每个点，计算其所在体素索引 (ix, iy, iz)
  3. 为每个新体素分配一个 voxel 编号，最多保留 max_voxels 个体素
  4. 在体素内部累积点，最多保留 max_points_per_voxel 个点
  5. 最终输出：
  6. voxels: (M, T, C)
  7. coords: (M, 4) [batch_idx, z, y, x]
  8. num_points: (M,)
输入¶

Args: points: (B, N, C) 批量点云数据： - B: batch size - N: 每帧点的最大数量 - C: 点特征维度（至少包含 x,y,z，通常为 4: [x,y,z,intensity]）。 voxel_size (tuple[float, float, float]): 体素大小 (vx, vy, vz)，单位与点云坐标一致。 point_cloud_range (tuple[float, float, float, float, float, float]): 点云范围 [x_min, y_min, z_min, x_max, y_max, z_max]。 max_points_per_voxel (int): 每个体素中允许保存的最大点数 T。 max_voxels (int): 每帧最多保留的体素数量 M_max。

输出¶

Returns: voxels: (M, T, C) 体素内点特征张量： - M: 实际生成的体素数量（M ≤ B * max_voxels） - T: 每个体素的最大点数，多余的点会被丢弃，未填满位置为 0。 coords: (M, 4) 每个体素在 3D 网格中的整数坐标 [batch_idx, z, y, x]。 num_points_per_voxel: (M,) 每个体素中真实点数，用于区分有效点与 padding。

模块用途¶
  • 将稠密点云转换为规则体素表示，作为后续 VFE 与 Sparse 3D Backbone 的输入
  • 作为链路 “LiDAR 原始点云 → LiDAR BEV 特征” 中的第一个离散化算子
  • 在算子框架中统一封装体素化逻辑，便于替换为更高性能的 C++/CUDA 实现

class LidarVoxelization(nn.Module):
    """
    LidarVoxelization 模块（点云体素化算子）。

    该模块将连续的 3D 点云坐标按照给定的 voxel_size 与 point_cloud_range
    量化到规则 3D 体素网格中，并输出体素中的点集合、体素坐标以及每个体素的点数。
    本实现为简化版 CPU 体素化逻辑，便于阅读与算子级重构，而非高性能实现。

    ---
    ### 结构说明（按顺序）
    1. 根据 point_cloud_range 与 voxel_size 计算 3D 网格大小 (nx, ny, nz)
    2. 遍历 batch 中每个点，计算其所在体素索引 (ix, iy, iz)
    3. 为每个新体素分配一个 voxel 编号，最多保留 max_voxels 个体素
    4. 在体素内部累积点，最多保留 max_points_per_voxel 个点
    5. 最终输出：
       - voxels:     (M, T, C)
       - coords:     (M, 4)  [batch_idx, z, y, x]
       - num_points: (M,)

    ---
    ### 输入
    Args:
        points: (B, N, C)
            批量点云数据：
              - B: batch size
              - N: 每帧点的最大数量
              - C: 点特征维度（至少包含 x,y,z，通常为 4: [x,y,z,intensity]）。
        voxel_size (tuple[float, float, float]):
            体素大小 (vx, vy, vz)，单位与点云坐标一致。
        point_cloud_range (tuple[float, float, float, float, float, float]):
            点云范围 [x_min, y_min, z_min, x_max, y_max, z_max]。
        max_points_per_voxel (int):
            每个体素中允许保存的最大点数 T。
        max_voxels (int):
            每帧最多保留的体素数量 M_max。

    ---
    ### 输出
    Returns:
        voxels: (M, T, C)
            体素内点特征张量：
              - M: 实际生成的体素数量（M ≤ B * max_voxels）
              - T: 每个体素的最大点数，多余的点会被丢弃，未填满位置为 0。
        coords: (M, 4)
            每个体素在 3D 网格中的整数坐标 [batch_idx, z, y, x]。
        num_points_per_voxel: (M,)
            每个体素中真实点数，用于区分有效点与 padding。

    ---
    ### 模块用途
    - 将稠密点云转换为规则体素表示，作为后续 VFE 与 Sparse 3D Backbone 的输入
    - 作为链路 “LiDAR 原始点云 → LiDAR BEV 特征” 中的第一个离散化算子
    - 在算子框架中统一封装体素化逻辑，便于替换为更高性能的 C++/CUDA 实现
    """

    def __init__(
        self,
        voxel_size: tuple,
        point_cloud_range: tuple,
        max_points_per_voxel: int = 32,
        max_voxels: int = 20000,
    ):
        super().__init__()  # 调用父类构造

        self.voxel_size = voxel_size                          # 记录体素大小
        self.point_cloud_range = point_cloud_range            # 记录点云范围
        self.max_points_per_voxel = max_points_per_voxel      # 每个体素最大点数
        self.max_voxels = max_voxels                          # 每帧最大体素数

        # 预计算网格大小 (nx, ny, nz)
        x_min, y_min, z_min, x_max, y_max, z_max = point_cloud_range  # 解包范围
        vx, vy, vz = voxel_size                                       # 解包体素尺寸

        nx = int((x_max - x_min) / vx + 1e-6)                         # x 方向体素数
        ny = int((y_max - y_min) / vy + 1e-6)                         # y 方向体素数
        nz = int((z_max - z_min) / vz + 1e-6)                         # z 方向体素数

        self.grid_size = (nx, ny, nz)                                 # 保存网格尺寸

    def forward(self, points: torch.Tensor):
        """
        Args:
            points (torch.Tensor):
                批量点云张量，形状为 (B, N, C)。      # 输入点云

        Returns:
            voxels (torch.Tensor):
                体素内点特征，形状 (M, T, C)。        # 输出体素
            coords (torch.Tensor):
                体素坐标，形状 (M, 4)。               # [batch, z, y, x]
            num_points_per_voxel (torch.Tensor):
                每个体素真实点数，形状 (M,)。         # 点数统计
        """
        assert points.dim() == 3, "points 必须是 (B, N, C) 张量"           # 基本检查

        B, N, C = points.shape                                          # 读取维度
        device = points.device                                          # 获取设备

        vx, vy, vz = self.voxel_size                                    # 体素大小
        x_min, y_min, z_min, x_max, y_max, z_max = self.point_cloud_range  # 点云范围
        nx, ny, nz = self.grid_size                                     # 网格大小

        # 最大体素总数：每帧 max_voxels，共 B 帧
        max_total_voxels = B * self.max_voxels                          # 体素总上限

        # 预分配输出张量
        voxels = torch.zeros(
            (max_total_voxels, self.max_points_per_voxel, C),
            dtype=points.dtype,
            device=device,
        )                                                                # 体素点特征
        coords = torch.zeros(
            (max_total_voxels, 4), dtype=torch.int32, device=device
        )                                                                # 体素坐标
        num_points_per_voxel = torch.zeros(
            (max_total_voxels,), dtype=torch.int32, device=device
        )                                                                # 每个体素点数

        voxel_count = 0                                                  # 当前体素计数
        voxel_map: dict = {}                                             # (b,z,y,x) → voxel_id 映射

        # 遍历 batch 中每一帧点云
        for b in range(B):                                               # 遍历每个 batch
            pts = points[b]                                              # (N,C)
            for n in range(N):                                           # 遍历每个点
                x, y, z = pts[n, 0].item(), pts[n, 1].item(), pts[n, 2].item()  # 读取坐标

                # 过滤超出范围的点
                if not (x_min <= x < x_max and
                        y_min <= y < y_max and
                        z_min <= z < z_max):
                    continue                                             # 跳过无效点

                # 计算体素索引
                ix = int((x - x_min) / vx)
                iy = int((y - y_min) / vy)
                iz = int((z - z_min) / vz)

                if not (0 <= ix < nx and 0 <= iy < ny and 0 <= iz < nz):
                    continue                                             # 再次安全检查

                key = (b, iz, iy, ix)                                    # 体素键值

                # 为新体素分配索引
                if key not in voxel_map:
                    if voxel_count >= max_total_voxels:                  # 超出体素总上限
                        continue
                    voxel_map[key] = voxel_count                         # 记录映射
                    coords[voxel_count] = torch.tensor(
                        [b, iz, iy, ix], dtype=torch.int32, device=device
                    )                                                    # 写入体素坐标
                    voxel_count += 1                                     # 体素计数加一

                voxel_id = voxel_map[key]                                # 获取体素索引

                # 若该体素已满，则丢弃多余点
                cur_num = num_points_per_voxel[voxel_id].item()          # 当前点数
                if cur_num >= self.max_points_per_voxel:
                    continue

                voxels[voxel_id, cur_num] = pts[n]                       # 写入点特征
                num_points_per_voxel[voxel_id] = cur_num + 1             # 点数加一

        # 截取到实际使用的体素数量
        voxels = voxels[:voxel_count]                                    # 裁剪多余体素
        coords = coords[:voxel_count]                                    # 裁剪坐标
        num_points_per_voxel = num_points_per_voxel[:voxel_count]        # 裁剪点数

        return voxels, coords, num_points_per_voxel                      # 返回体素化结果

'VoxelFeatureEncoder'¶

对体素中的点云进行特征编码。

VoxelFeatureEncoder ¶

Bases: Module

VoxelFeatureEncoder 模块（体素特征编码算子，VFE）。

该模块接收体素化后的点云数据，通过逐点 MLP 编码并在体素内部做 聚合（如 max pooling），将每个体素内不定数量的点特征压缩为一个 定长向量，用作后续稀疏 3D 主干网络的输入。

结构说明（按顺序）¶
  1. 对体素内每个点做逐点特征编码： Linear C_in → C_mid + BatchNorm1d + ReLU
  2. 在体素内部沿点维度做 max pooling： 将 (M, T, C_mid) 聚合为 (M, C_mid)
  3. 可选再接一层线性变换： Linear C_mid → C_out + BatchNorm1d + ReLU
输入¶

Args: voxels: (M, T, C_in) 体素内点特征： - M: 体素数量 - T: 每个体素的最大点数 - C_in: 点特征维度（如 [x,y,z,intensity,...]）。 num_points_per_voxel: (M,) 每个体素中真实点数，用于区分有效点与 padding。

输出¶

Returns: voxel_feats: (M, C_out) 编码后的体素特征，每个体素对应一个定长向量。

模块用途¶
  • 将不定点数的体素内点云压缩为固定维度特征
  • 作为链路 “LiDAR 体素化 → 稀疏 3D 主干网络” 的接口算子
  • 可以替换为更复杂的多层 VFE 或带残差的体素编码器

class VoxelFeatureEncoder(nn.Module):
    """
    VoxelFeatureEncoder 模块（体素特征编码算子，VFE）。

    该模块接收体素化后的点云数据，通过逐点 MLP 编码并在体素内部做
    聚合（如 max pooling），将每个体素内不定数量的点特征压缩为一个
    定长向量，用作后续稀疏 3D 主干网络的输入。

    ---
    ### 结构说明（按顺序）
    1. 对体素内每个点做逐点特征编码：
       **Linear C_in → C_mid + BatchNorm1d + ReLU**
    2. 在体素内部沿点维度做 **max pooling**：
       将 (M, T, C_mid) 聚合为 (M, C_mid)
    3. 可选再接一层线性变换：
       **Linear C_mid → C_out + BatchNorm1d + ReLU**

    ---
    ### 输入
    Args:
        voxels: (M, T, C_in)
            体素内点特征：
              - M: 体素数量
              - T: 每个体素的最大点数
              - C_in: 点特征维度（如 [x,y,z,intensity,...]）。
        num_points_per_voxel: (M,)
            每个体素中真实点数，用于区分有效点与 padding。

    ---
    ### 输出
    Returns:
        voxel_feats: (M, C_out)
            编码后的体素特征，每个体素对应一个定长向量。

    ---
    ### 模块用途
    - 将不定点数的体素内点云压缩为固定维度特征
    - 作为链路 “LiDAR 体素化 → 稀疏 3D 主干网络” 的接口算子
    - 可以替换为更复杂的多层 VFE 或带残差的体素编码器
    """

    def __init__(
        self,
        in_channels: int,
        mid_channels: int,
        out_channels: int,
        use_second_linear: bool = True,
    ):
        """
        Args:
            in_channels (int):
                输入点特征维度 C_in。                 # 点特征通道
            mid_channels (int):
                中间编码通道数 C_mid。                # 中间通道
            out_channels (int):
                输出体素特征维度 C_out。              # 输出通道
            use_second_linear (bool):
                是否使用第二层 Linear+CBN+ReLU。     # 控制是否使用第二层 MLP
        """
        super().__init__()  # 父类构造

        # 第一层逐点编码：Linear + BN + ReLU
        self.linear1 = nn.Linear(in_channels, mid_channels, bias=False)  # 线性映射
        self.bn1 = nn.BatchNorm1d(mid_channels)                          # BN
        self.use_second_linear = use_second_linear                       # 记录配置

        if use_second_linear:
            # 第二层逐点 / 逐体素编码
            self.linear2 = nn.Linear(mid_channels, out_channels, bias=False)  # 线性映射
            self.bn2 = nn.BatchNorm1d(out_channels)                           # BN
            self.out_channels = out_channels                                  # 输出维度
        else:
            self.out_channels = mid_channels                                  # 输出维度等于中间层

    def forward(
        self,
        voxels: torch.Tensor,
        num_points_per_voxel: torch.Tensor,
    ) -> torch.Tensor:
        """
        Args:
            voxels (torch.Tensor):
                体素内点特征，形状 (M, T, C_in)。         # 输入体素
            num_points_per_voxel (torch.Tensor):
                每个体素真实点数，形状 (M,)。             # 有效点数

        Returns:
            torch.Tensor:
                编码后的体素特征，形状 (M, C_out)。       # 输出体素特征
        """
        assert voxels.dim() == 3, "voxels 必须是 (M, T, C_in) 张量"                # 检查维度
        assert num_points_per_voxel.dim() == 1, "num_points_per_voxel 必须是一维张量"  # 检查维度

        M, T, C_in = voxels.shape                                               # 读取尺寸
        x = voxels.reshape(M * T, C_in)                                         # 合并体素和点维度

        # 逐点编码：Linear1 + BN1 + ReLU
        x = self.linear1(x)                                                     # 线性变换
        x = self.bn1(x)                                                         # BN
        x = F.relu(x, inplace=True)                                             # 激活

        x = x.reshape(M, T, -1)                                                 # 恢复为 (M,T,C_mid)

        # 体素内部 max pooling（掩码无效点）
        mask = (
            torch.arange(T, device=voxels.device)[None, :].expand(M, T)
            < num_points_per_voxel.unsqueeze(1)
        )                                                                       # 有效点掩码 (M,T)
        mask = mask.unsqueeze(-1)                                               # (M,T,1)

        x_masked = x.masked_fill(~mask, float("-inf"))                          # 无效位置置为 -inf
        voxel_feats, _ = x_masked.max(dim=1)                                    # 沿点维度取最大值 → (M,C_mid)

        voxel_feats[voxel_feats == float("-inf")] = 0.0                         # 处理全无效情况

        if self.use_second_linear:
            # 第二层编码：Linear2 + BN2 + ReLU
            voxel_feats = self.linear2(voxel_feats)                             # 线性变换
            voxel_feats = self.bn2(voxel_feats)                                 # BN
            voxel_feats = F.relu(voxel_feats, inplace=True)                     # 激活

        return voxel_feats                                                      # 返回体素特征

class Sparse3DBackbone(nn.Module):
    """
    Sparse3DBackbone 模块（稀疏三维主干网络算子 / SECOND 风格简化版）。

    该模块接收体素特征及其三维坐标，将稀疏体素特征嵌入到规则 3D 体素网格中，
    再通过 3D 卷积对整个体素网格进行特征提取，输出形状为 (B, C_out, Z, Y, X)
    的三维特征体，供后续高度维压缩算子使用。

    注意：本实现为“稠密 3D 卷积版本”的简化骨架，用于算子级重构与实验 Demo，
    并非真正的高性能 SparseConv 实现。

    Args:
        in_channels: int
            输入体素特征通道数 C_in。
        hidden_channels: int
            中间层通道数 C_mid，用于增强表达能力。
        out_channels: int
            输出三维特征通道数 C_out。
        grid_size: tuple[int, int, int]
            三维体素网格尺寸 (Z, Y, X)，与体素化阶段使用的网格大小一致。

    Returns:
        feat_3d: (B, C_out, Z, Y, X)
            编码后的三维体素特征体，用于后续 3D→BEV 压缩算子。
    """

    def __init__(
        self,
        in_channels: int,
        hidden_channels: int,
        out_channels: int,
        grid_size: Tuple[int, int, int],
    ):
        super().__init__()                                             # 调用父类构造

        self.in_channels = in_channels                                 # 输入通道数
        self.hidden_channels = hidden_channels                         # 中间通道数
        self.out_channels = out_channels                               # 输出通道数
        self.grid_size = grid_size                                     # (Z, Y, X) 网格大小

        # 简化版 3D 主干：两层 3D 卷积 + BN + ReLU
        self.conv_block1 = nn.Sequential(
            nn.Conv3d(in_channels, hidden_channels, kernel_size=3, padding=1, bias=False),  # 3D 卷积
            nn.BatchNorm3d(hidden_channels),                                                # BN
            nn.ReLU(inplace=True),                                                         # 激活
        )

        self.conv_block2 = nn.Sequential(
            nn.Conv3d(hidden_channels, out_channels, kernel_size=3, padding=1, bias=False),  # 3D 卷积
            nn.BatchNorm3d(out_channels),                                                    # BN
            nn.ReLU(inplace=True),                                                          # 激活
        )

    def forward(
        self,
        voxel_feats: torch.Tensor,
        voxel_coords: torch.Tensor,
        batch_size: int,
    ) -> torch.Tensor:
        """
        Args:
            voxel_feats (torch.Tensor):
                体素特征，形状 (M, C_in)。              # 每个体素一个特征向量
            voxel_coords (torch.Tensor):
                体素坐标，形状 (M, 4)，格式为
                [batch_idx, z, y, x]。                 # 体素在网格中的索引
            batch_size (int):
                当前批次大小 B。                       # batch 数量

        Returns:
            torch.Tensor:
                三维特征体，形状 (B, C_out, Z, Y, X)。  # 稠密 3D 特征
        """
        assert voxel_feats.dim() == 2, "voxel_feats 必须是 (M, C_in) 张量"       # 检查维度
        assert voxel_coords.dim() == 2 and voxel_coords.shape[1] == 4, "voxel_coords 必须是 (M, 4) 张量"  # 检查维度

        device = voxel_feats.device                                           # 获取设备
        M, C_in = voxel_feats.shape                                           # 读取体素数量与通道数
        Z, Y, X = self.grid_size                                              # 解包网格尺寸

        # 初始化稠密体素网格： (B, C_in, Z, Y, X)
        dense_grid = torch.zeros(
            batch_size,
            C_in,
            Z,
            Y,
            X,
            dtype=voxel_feats.dtype,
            device=device,
        )                                                                      # 初始化 3D 体素网格

        # 将稀疏体素特征 scatter 到稠密体素网格中
        b = voxel_coords[:, 0].long()                                         # batch 索引
        z = voxel_coords[:, 1].long()                                         # z 轴索引
        y = voxel_coords[:, 2].long()                                         # y 轴索引
        x = voxel_coords[:, 3].long()                                         # x 轴索引

        # 过滤越界体素（理论上 voxelization 已经保证合法，这里再做一次安全检查）
        valid_mask = (
            (b >= 0) & (b < batch_size) &
            (z >= 0) & (z < Z) &
            (y >= 0) & (y < Y) &
            (x >= 0) & (x < X)
        )                                                                      # 合法体素掩码

        if valid_mask.any():                                                  # 若存在合法体素
            b_valid = b[valid_mask]                                           # 有效 batch 索引
            z_valid = z[valid_mask]                                           # 有效 z 索引
            y_valid = y[valid_mask]                                           # 有效 y 索引
            x_valid = x[valid_mask]                                           # 有效 x 索引
            feats_valid = voxel_feats[valid_mask]                             # 有效体素特征 (M_valid,C_in)

            dense_grid[b_valid, :, z_valid, y_valid, x_valid] = feats_valid   # 填充到稠密网格中

        # 通过 3D 卷积主干提取三维特征
        x3d = self.conv_block1(dense_grid)                                    # 第一层 3D 卷积块
        x3d = self.conv_block2(x3d)                                           # 第二层 3D 卷积块

        return x3d                                                            # 返回三维特征体

        dense_grid[b_valid, :, z_valid, y_valid, x_valid] = feats_valid   # 填充到稠密网格中

    x3d = self.conv_block1(dense_grid)                                    # 第一层 3D 卷积块
    x3d = self.conv_block2(x3d)                                           # 第二层 3D 卷积块

    return x3d                                                            # 返回三维特征体

@torch.library.custom_op(
    "bevfusion_ops::lidar_flatten_z_to_bev",
    mutates_args=[]
)
def lidar_flatten_z_to_bev(
    feat_3d: torch.Tensor,
) -> torch.Tensor:
    """
    将 LiDAR 三维体素特征沿高度维度 z 进行聚合，压缩为 BEV 平面上的
    二维特征图，输出形状为 (B, C, H, W)。

    Args:
        feat_3d: (B, C, Z, H, W)
            LiDAR 分支输出的三维特征体：
              - B: batch size
              - C: 通道数
              - Z: 高度方向体素层数
              - H, W: 水平平面上的体素网格尺寸。

    Returns:
        bev_feat: (B, C, H, W)
            沿高度维度聚合后的 BEV 特征图，这里采用 sum pooling 进行聚合。
    """
    assert feat_3d.dim() == 5, "feat_3d 必须是 (B, C, Z, H, W) 张量"   # 基本形状检查

    # 使用 sum pooling 沿高度维度聚合：
    # (B, C, Z, H, W) → (B, C, H, W)
    bev_feat = feat_3d.sum(dim=2)                                      # 沿 Z 维求和

    return bev_feat                                                    # 返回 BEV 特征
