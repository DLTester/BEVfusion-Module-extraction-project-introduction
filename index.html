BEVFusion 模块化提取文档¶

本项目对 BEVFusion 网络的关键组件进行结构化拆分，形成 可复用算子（Operators） 与 可插拔网络模块（Networks），方便理解、调试与二次开发。

本文档旨在提供：

  • 各子模块的功能说明
  • 对应算子与网络层的 API 文档
  • 主框架 BEVFusionFramework 的整体调用关系

文档主要分为以下几个部分：

📷 Camera 侧模块¶

包括图像特征提取、几何变换以及将特征映射到 BEV 空间的相关算子与网络结构。

详见：
👉 camera_side.md

🛰 LiDAR 侧模块¶

包含体素化、稀疏卷积 Backbone、点云 BEV 映射等核心模块。

详见：
👉 lidar_side.md

🗺 BEV 空间融合模块¶

描述图像 BEV 与点云 BEV 的融合流程，包括 BEV 编码器与融合算子。

详见：
👉 bev_space_integration.md

🧩 BEVFusion 主框架¶

为以上各模块提供统一封装，定义完整的 BEVFusion 推理流程。

详见：
👉 BEVFusionFramework.md
