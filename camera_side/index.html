Camera Side（图像侧模块）¶

Camera Side 模块的作用是：
从多视角图像中提取语义特征、深度信息，并通过几何方法将 2D 特征转换到 3D / BEV 空间，为后续 BEV 融合提供图像侧 BEV 特征。

本模块包含四类核心功能：

  1. 图像特征提取网络（FPN / Backbone）
    提取多尺度图像语义特征。

  2. 深度预测模块
    输出像素级深度概率分布，用于 Lift 特征到 3D。

  3. 几何投影算子（Lift / Transform / Pooling）
    将图像特征映射至 3D 空间，再投影到 BEV 网格。

  4. 特征增强算子
    对图像 BEV 特征进行加权或扩展。

以下列出 Camera Side 中全部算子与网络模块，并给出其对应文件路径。

CameraFPNFusion¶

多尺度 FPN 特征融合模块。

CameraFPNFusion ¶

Bases: Module

CameraFPNFusion 模块（图像多尺度特征融合算子）。

该模块通过对多尺度相机特征分别做 1×1 Conv 通道映射、上采样到统一空间分辨率， 然后在通道维度上逐元素相加，得到一张融合后的 2D 特征图。该特征图作为后续 深度估计与 Camera-to-BEV 视角变换的统一输入接口。

结构说明（按顺序）¶
  1. 对每个尺度特征使用 Conv2d C_in_i → C_out, kernel_size=1, stride=1, padding=0
  2. 做通道线性变换，映射到统一通道数 C_out
  3. (可选) BatchNorm2d
  4. 对每个尺度映射后的特征做归一化
  5. (可选) ReLU 激活
  6. 在融合前增强非线性表达能力
  7. 特征上采样到最高分辨率并逐元素相加
  8. 得到单尺度、多通道的融合特征图
输入¶

Args: feats (List[torch.Tensor]): 多尺度相机特征列表，长度为 N_scales。 第 i 个特征形状为 (B, C_in_i, H_i, W_i)： - B: batch size - C_in_i: 第 i 个尺度的通道数 - H_i, W_i: 第 i 个尺度的空间尺寸

输出¶

Returns: torch.Tensor: 融合后的单尺度特征图，形状为 (B, C_out, H_out, W_out)， 其中 H_out, W_out 等于列表中最高分辨率特征图的空间尺寸。

模块用途¶
  • 将来自相机 backbone 的多尺度特征对齐到统一空间尺度与通道数
  • 作为后续 depth head / LSS 视角变换算子的统一 2D 特征输入
  • 也可作为通用的多尺度特征融合算子复用于其它相机分支网络

class CameraFPNFusion(nn.Module):
    """
    CameraFPNFusion 模块（图像多尺度特征融合算子）。

    该模块通过对多尺度相机特征分别做 1×1 Conv 通道映射、上采样到统一空间分辨率，
    然后在通道维度上逐元素相加，得到一张融合后的 2D 特征图。该特征图作为后续
    深度估计与 Camera-to-BEV 视角变换的统一输入接口。

    ---
    ### 结构说明（按顺序）
    1. 对每个尺度特征使用 **Conv2d C_in_i → C_out, kernel_size=1, stride=1, padding=0**
       - 做通道线性变换，映射到统一通道数 C_out
    2. **(可选) BatchNorm2d**
       - 对每个尺度映射后的特征做归一化
    3. **(可选) ReLU 激活**
       - 在融合前增强非线性表达能力
    4. 特征上采样到最高分辨率并逐元素相加
       - 得到单尺度、多通道的融合特征图

    ---
    ### 输入
    Args:
        feats (List[torch.Tensor]):
            多尺度相机特征列表，长度为 N_scales。
            第 i 个特征形状为 (B, C_in_i, H_i, W_i)：
              - B: batch size
              - C_in_i: 第 i 个尺度的通道数
              - H_i, W_i: 第 i 个尺度的空间尺寸

    ---
    ### 输出
    Returns:
        torch.Tensor:
            融合后的单尺度特征图，形状为 (B, C_out, H_out, W_out)，
            其中 H_out, W_out 等于列表中最高分辨率特征图的空间尺寸。

    ---
    ### 模块用途
    - 将来自相机 backbone 的多尺度特征对齐到统一空间尺度与通道数
    - 作为后续 depth head / LSS 视角变换算子的统一 2D 特征输入
    - 也可作为通用的多尺度特征融合算子复用于其它相机分支网络
    """

    def __init__(
        self,
        in_channels_list,
        out_channels: int,
        use_bn: bool = True,
        use_relu: bool = True,
    ):
        """
        Args:
            in_channels_list (List[int]):
                多尺度输入特征的通道数列表，长度为 N_scales。  # 简短解释
            out_channels (int):
                融合后特征的目标通道数。                        # 目标通道
            use_bn (bool):
                是否在 1×1 Conv 后使用 BatchNorm2d。            # 控制是否使用 BN
            use_relu (bool):
                是否在 BN 后使用 ReLU 激活。                     # 控制是否使用 ReLU
        """
        super().__init__()  # 调用父类构造

        self.out_channels = out_channels  # 记录输出通道数
        self.use_bn = use_bn              # 记录是否使用 BN
        self.use_relu = use_relu          # 记录是否使用 ReLU

        # 为每个尺度构建一套 1×1 Conv (+BN+ReLU) 进行通道映射
        self.proj_layers = nn.ModuleList()           # 保存每个尺度的映射层
        for c_in in in_channels_list:                # 遍历每个尺度的输入通道
            layers = [nn.Conv2d(c_in, out_channels, 1, bias=not use_bn)]  # 1×1 Conv
            if use_bn:
                layers.append(nn.BatchNorm2d(out_channels))               # 可选 BN
            if use_relu:
                layers.append(nn.ReLU(inplace=True))                      # 可选 ReLU
            self.proj_layers.append(nn.Sequential(*layers))               # 封装为顺序模块

    def forward(self, feats):
        """
        前向计算：将多尺度特征映射到统一通道并上采样后相加。

        Args:
            feats (List[torch.Tensor]):
                多尺度相机特征列表，形状参见类注释。           # 输入特征列表

        Returns:
            torch.Tensor:
                融合后的单尺度特征图 (B, C_out, H_out, W_out)。  # 返回融合特征
        """
        assert len(feats) == len(self.proj_layers), "feats 数量与 proj_layers 不一致"  # 简单检查

        # 以第一个特征为最高分辨率参考
        ref_feat = feats[0]                                               # 参考尺度特征
        _, _, H_ref, W_ref = ref_feat.shape                               # 参考空间尺寸

        fused = 0.0                                                       # 初始化融合结果
        for feat, proj in zip(feats, self.proj_layers):                   # 遍历每个尺度
            x = proj(feat)                                                # 通道映射到 C_out
            if x.shape[-2:] != (H_ref, W_ref):                            # 若空间尺寸不同
                x = F.interpolate(x, size=(H_ref, W_ref),
                                   mode="bilinear", align_corners=False)  # 上采样到参考尺度
            fused = fused + x                                             # 逐尺度累加特征

        return fused                                                      # 返回融合结果

DepthDistributionHead¶

像素级深度分布预测模块。

DepthDistributionHead ¶

Bases: Module

DepthDistributionHead 模块（像素深度分布预测算子）。

该模块接收融合后的相机 2D 特征图，通过若干 2D 卷积层预测每个像素在 离散深度 bins 上的概率分布，用于后续的 2D→3D 升维与 Camera-to-BEV 视角变换。输出为 (B, D, H, W)，在深度维度上做 softmax 归一化。

结构说明（按顺序）¶
  1. Conv2d C_in → C_mid, kernel_size=3, padding=1
  2. 对输入特征做局部感受野增强
  3. (可选) BatchNorm2d + ReLU
  4. 稳定训练并引入非线性
  5. Conv2d C_mid → D, kernel_size=1
  6. 映射到深度通道维度 D
  7. Softmax (沿 depth 维度)
  8. 得到每个像素的深度概率分布
输入¶

Args: x (torch.Tensor): 相机融合特征，形状 (B, C_in, H, W)： - B: batch size - C_in: 输入通道数 - H, W: 特征图空间尺寸

输出¶

Returns: torch.Tensor: 像素深度概率分布，形状 (B, D, H, W)，在维度 1 上求和为 1。

模块用途¶
  • 为 LSS / BEVFusion 提供每个像素在离散深度上的概率估计
  • 后续算子 camera_lift_2d_to_3d、depth_weighted_feature_expand 将复用该深度分布完成 2D→3D 升维与特征加权

class DepthDistributionHead(nn.Module):
    """
    DepthDistributionHead 模块（像素深度分布预测算子）。

    该模块接收融合后的相机 2D 特征图，通过若干 2D 卷积层预测每个像素在
    离散深度 bins 上的概率分布，用于后续的 2D→3D 升维与 Camera-to-BEV
    视角变换。输出为 (B, D, H, W)，在深度维度上做 softmax 归一化。

    ---
    ### 结构说明（按顺序）
    1. **Conv2d C_in → C_mid, kernel_size=3, padding=1**
       - 对输入特征做局部感受野增强
    2. **(可选) BatchNorm2d + ReLU**
       - 稳定训练并引入非线性
    3. **Conv2d C_mid → D, kernel_size=1**
       - 映射到深度通道维度 D
    4. **Softmax (沿 depth 维度)**
       - 得到每个像素的深度概率分布

    ---
    ### 输入
    Args:
        x (torch.Tensor):
            相机融合特征，形状 (B, C_in, H, W)：
              - B: batch size
              - C_in: 输入通道数
              - H, W: 特征图空间尺寸

    ---
    ### 输出
    Returns:
        torch.Tensor:
            像素深度概率分布，形状 (B, D, H, W)，在维度 1 上求和为 1。

    ---
    ### 模块用途
    - 为 LSS / BEVFusion 提供每个像素在离散深度上的概率估计
    - 后续算子 `camera_lift_2d_to_3d`、`depth_weighted_feature_expand`
      将复用该深度分布完成 2D→3D 升维与特征加权
    """

    def __init__(
        self,
        in_channels: int,
        mid_channels: int,
        num_depth_bins: int,
        use_bn: bool = True,
    ):
        """
        Args:
            in_channels (int):
                输入特征通道数。                       # C_in
            mid_channels (int):
                中间隐层通道数。                       # C_mid
            num_depth_bins (int):
                深度离散 bin 数量 D。                  # D
            use_bn (bool):
                是否在第一层卷积后使用 BatchNorm2d。  # 控制是否使用 BN
        """
        super().__init__()                                               # 父类构造

        layers = [                                                       # 第一层 conv 及可选 BN+ReLU
            nn.Conv2d(in_channels, mid_channels, kernel_size=3, padding=1, bias=not use_bn)  # 3×3 Conv
        ]
        if use_bn:                                                       # 是否加入 BN
            layers.append(nn.BatchNorm2d(mid_channels))                  # BN 层
        layers.append(nn.ReLU(inplace=True))                             # ReLU 激活

        self.stem = nn.Sequential(*layers)                               # 封装为顺序模块
        self.depth_logits = nn.Conv2d(mid_channels, num_depth_bins, kernel_size=1)  # 1×1 Conv 预测 D 通道

        self.num_depth_bins = num_depth_bins                             # 记录深度 bins 数

    def forward(self, x: torch.Tensor) -> torch.Tensor:
        """
        Args:
            x (torch.Tensor):
                输入相机特征，形状 (B, C_in, H, W)。   # 输入特征

        Returns:
            torch.Tensor:
                深度概率分布 (B, D, H, W)。           # 输出分布
        """
        feat = self.stem(x)                                              # 提取中间特征
        logits = self.depth_logits(feat)                                 # 生成深度 logits

        # 在深度维度上做 softmax，得到每个像素的深度概率分布
        prob = F.softmax(logits, dim=1)                                  # 沿通道维归一化

        return prob                                                      # 返回深度分布

        nn.Conv2d(in_channels, mid_channels, kernel_size=3, padding=1, bias=not use_bn)  # 3×3 Conv
    ]
    if use_bn:                                                       # 是否加入 BN
        layers.append(nn.BatchNorm2d(mid_channels))                  # BN 层
    layers.append(nn.ReLU(inplace=True))                             # ReLU 激活

    self.stem = nn.Sequential(*layers)                               # 封装为顺序模块
    self.depth_logits = nn.Conv2d(mid_channels, num_depth_bins, kernel_size=1)  # 1×1 Conv 预测 D 通道

    self.num_depth_bins = num_depth_bins                             # 记录深度 bins 数

    mutates_args=[]
)
def camera_bev_pooling(
    feat_3d: torch.Tensor,
    bev_indices: torch.Tensor,
    bev_h: int,
    bev_w: int,
) -> torch.Tensor:
    """
    将相机分支在体素 / frustum 空间中的三维特征聚合到 BEV 网格上，
    得到形状为 (B, C, bev_h, bev_w) 的 Camera BEV 特征图。

    Args:
        feat_3d: (B, D, C, H, W)
            三维特征张量：
              - B: batch size
              - D: 深度维度长度（如深度 bins 或体素层数）
              - C: 通道数
              - H, W: 图像平面上的采样网格尺寸。
        bev_indices: (B, D, H, W, 2)
            每个三维体素在 BEV 网格上的整数索引 (i, j)：
              - i: BEV 中的纵向索引，范围 [0, bev_h - 1]
              - j: BEV 中的横向索引，范围 [0, bev_w - 1]
              - 若某个位置为无效点，可将对应索引置为负数（如 -1）。
        bev_h: int
            BEV 网格的高度（纵向网格数）。
        bev_w: int
            BEV 网格的宽度（横向网格数）。

    Returns:
        bev_feat: (B, C, bev_h, bev_w)
            在 BEV 网格上聚合后的相机特征图，默认使用求和聚合（sum pooling）。
    """
    assert feat_3d.dim() == 5, "feat_3d 必须是 (B, D, C, H, W) 张量"                    # 基本形状检查
    assert bev_indices.dim() == 5, "bev_indices 必须是 (B, D, H, W, 2) 张量"            # 基本形状检查

    B, D, C, H, W = feat_3d.shape                                                     # 读取输入尺寸
    assert bev_indices.shape[0] == B and bev_indices.shape[1] == D, "batch / depth 维度不匹配"  # 检查 B,D
    assert bev_indices.shape[2] == H and bev_indices.shape[3] == W, "空间尺寸 (H,W) 不匹配"      # 检查 H,W

    device = feat_3d.device                                                           # 获取设备
    bev_feat = torch.zeros(B, C, bev_h, bev_w, device=device)                         # 初始化输出 BEV 特征

    # 将三维体素展平到一个列表，按 batch 分别做 scatter-add
    # feat_3d: (B, D, C, H, W) → (B, N, C)，其中 N = D*H*W
    N = D * H * W                                                                      # 三维位置数量
    feat_flat = feat_3d.permute(0, 1, 3, 4, 2).reshape(B, N, C)                        # (B,N,C)
    idx_flat = bev_indices.reshape(B, N, 2)                                            # (B,N,2)

    for b in range(B):                                                                 # 遍历每个 batch
        feat_b = feat_flat[b]                                                          # (N,C)
        idx_b = idx_flat[b]                                                            # (N,2)

        i = idx_b[:, 0]                                                                # 纵向索引
        j = idx_b[:, 1]                                                                # 横向索引

        # 有效位置掩码：索引在合法范围内
        valid_mask = (i >= 0) & (j >= 0) & (i < bev_h) & (j < bev_w)                   # 过滤无效点
        if not torch.any(valid_mask):                                                  # 若全部无效则跳过
            continue

        i_valid = i[valid_mask]                                                        # 有效 i
        j_valid = j[valid_mask]                                                        # 有效 j
        feat_valid = feat_b[valid_mask]                                                # 有效特征 (M,C)

        linear_idx = (i_valid * bev_w + j_valid).long()                                # 映射到一维索引 (M,)

        # 将特征累加到 BEV 网格上：先展平为 (C, bev_h*bev_w)，在列维度做 index_add_
        bev_b_flat = bev_feat[b].reshape(C, bev_h * bev_w)                             # (C, bev_h*bev_w)
        bev_b_flat.index_add_(                                                         # 按列累加特征
            dim=1,
            index=linear_idx,
            source=feat_valid.T,                                                       # (C,M)
        )

    return bev_feat                                                                    # 返回聚合后的 BEV 特征

def camera_lift_2d_to_3d(
    pixels_2d: torch.Tensor,
    depths: torch.Tensor,
    K: torch.Tensor,
    R: torch.Tensor,
    t: torch.Tensor,
) -> torch.Tensor:
    """
    计算 2D 像素坐标在给定离散深度下对应的 3D 空间坐标，并返回形状为 (D, H, W, 3) 的三维点云张量。

    Args:
        pixels_2d: (H, W, 2)
            像素坐标网格，格式为 (u, v)，表示图像平面上的二维像素坐标。
        depths: (D,)
            离散深度取样值，一维深度 bin 向量。
        K: (3, 3)
            相机内参矩阵。
        R: (3, 3)
            相机到目标坐标系的旋转矩阵。
        t: (3,)
            相机到目标坐标系的平移向量。

    Returns:
        points_3d: (D, H, W, 3)
            每个像素在每个深度 bin 下对应的三维空间坐标 (x, y, z)。
    """
    H, W, _ = pixels_2d.shape                        # 图像高宽
    D = depths.shape[0]                              # 深度 bin 数

    u = pixels_2d[..., 0].reshape(1, H, W)           # 提取 u 坐标
    v = pixels_2d[..., 1].reshape(1, H, W)           # 提取 v 坐标
    z = depths.view(D, 1, 1)                         # 深度扩展到 (D,1,1)

    # 由针孔成像模型反算相机坐标系下的 (x, y, z)
    x = (u - K[0, 2]) * z / K[0, 0]                  # 反投影得到 x
    y = (v - K[1, 2]) * z / K[1, 1]                  # 反投影得到 y
    z_expanded = z.expand_as(x)                      # 扩展 z 以便与 x,y 对齐

    pts_cam = torch.stack([x, y, z_expanded], dim=-1)  # 相机系 3D 点 (D,H,W,3)

    pts_flat = pts_cam.reshape(-1, 3)                # 展平为 (D*H*W, 3)
    pts_world = (pts_flat @ R.T) + t.view(1, 3)      # 坐标变换到目标坐标系
    pts_world = pts_world.reshape(D, H, W, 3)        # 恢复形状为 (D,H,W,3)

    return pts_world                                 # 返回三维点云

    """

    得到形状为 (B, D, C, H, W) 的三维特征张量。

    Args:
        img_feats: (B, C, H, W)
            相机分支输出的 2D 特征图：
              - B: batch size
              - C: 通道数
              - H, W: 特征图空间尺寸。
        depth_prob: (B, D, H, W)
            对应像素的离散深度概率分布：
              - D: 深度 bin 数量
              - 与 img_feats 在 (B, H, W) 维度上对齐。

    Returns:
        feat_3d: (B, D, C, H, W)
            按深度概率加权并在深度维度展开后的三维特征张量，
            每个体素位置同时编码了像素语义与对应深度权重。
    """
    assert img_feats.dim() == 4, "img_feats 必须是 (B, C, H, W) 张量"           # 简单形状检查
    assert depth_prob.dim() == 4, "depth_prob 必须是 (B, D, H, W) 张量"        # 简单形状检查
    assert img_feats.shape[0] == depth_prob.shape[0], "batch 维度不匹配"        # 检查 B
    assert img_feats.shape[2:] == depth_prob.shape[2:], "空间尺寸 (H,W) 不匹配"  # 检查 H,W

    # 扩展维度以便广播相乘：
    # img_feats:   (B,    C, H, W) → (B, 1, C, H, W)
    # depth_prob:  (B,    D, H, W) → (B, D, 1, H, W)
    img_feats_expanded = img_feats.unsqueeze(1)                                  # 在深度位置插入维度
    depth_prob_expanded = depth_prob.unsqueeze(2)                                # 在通道位置插入维度

    feat_3d = img_feats_expanded * depth_prob_expanded                           # 按深度概率加权展开

    return feat_3d                                                                # (B, D, C, H, W)
